# 레시피투유(RecipeToYou)
## 프로젝트 개요
> 온라인 시장 규모 확대와 건강 가정식에 대한 관심도 증가로 인한 온라인 커뮤니티 활성화가 된 현시대를 고려하여 온라인 쇼핑몰과 레시피 커뮤니트 웹 애플리케이션 구축을 목표로 한 'RecipeToYou' 프로젝트입니다.

+ 작업 기간 : 2021. 12. 20 ~ 2022. 03. 10
+ 참여 인원 : 7명
+ 사용 기술
